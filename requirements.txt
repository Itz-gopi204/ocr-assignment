# INSTALLATION INSTRUCTIONS
# ============================================
#
# 1. Create a virtual environment (recommended):
#    python -m venv ocr_env
#
# 2. Activate the environment:
#    Windows: ocr_env\Scripts\activate
#    Linux/Mac: source ocr_env/bin/activate
#
# 3. Install dependencies:
#    pip install -r requirements.txt
#
# 4. Install Tesseract OCR engine:
#    Windows: Download from https://github.com/UB-Mannheim/tesseract/wiki
#             Add to PATH: C:\Program Files\Tesseract-OCR
#    Linux: sudo apt-get install tesseract-ocr
#    Mac: brew install tesseract
#
# 5. Download spaCy model:
#    python -m spacy download en_core_web_sm
#
# 6. Run Options:
#    - Jupyter Notebook: jupyter notebook ocr_pii_pipeline.ipynb
#    - Streamlit App:    streamlit run streamlit_app.py
#    - Python Script:    python enhanced_pipeline.py
